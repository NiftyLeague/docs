乱闘する準備をしなさい！ ローカルマルチバージョン **[ニフティSmashers](https://nifty-league.com/games)** オンラインマルチ続い起動時にすぐに利用できるようになりました！ だから、いくつかのコントローラーをつかんで🎮そして友達と遊んでください。

コミュニティの中での戦いそれを、多くのとして取得 _バットbonks_ することができますようにあなたの友人🥖on！ **Nifty Smashers** は、古典的なスーパースマッシュブラザーズゲームからインスピレーションを得ており、その目的は、対戦相手をマップからノックアウトしてポイントを獲得することです。 プレイするには **** を選択し、キーボード、プレイステーション、またはXboxコントローラーを入力として使用する必要があります（ソファマルチプレイヤーも機能します！）。 ゲームロビーから、ウォレットで所有している以前に作成した **DEGENs** から選択して、クールな作品を披露することができます。 各部族には独自の特別な動きがあるので、さまざまな部族を試して、自分のプレイスタイルに最適なムーブセットを見つけてください。 チャージアップされたバットアタックの攻撃ボタンを押し続けると、いくつかの特別な動きもチャージされる可能性があります。 あなたが他のプレイヤーを攻撃するとき、彼らは1ポイントの価値があります。 連続してボンボンを打つほど、バウンドが速くなり、価値のあるポイントが増えます。 最後のメガボンクを着陸させてマップから外し、すべてのポイントを獲得しましょう！

**すでに述べたように、これは最初のゲームのベータリリースであるため、厄介なボットのラグと問題が予想されます。** 当面の計画は、 **Nifty Smashers** 開発を継続し、追加のコンボ移動や進行システムなどの機能をさらに導入することです。 **[Discord](https://discord.gg/niftyleague)** **Nifty Smashers** を改善し、それを次のレベルに引き上げる方法に関するフィードバックとアイデアを提供してください。

**オンラインマルチプレイヤーラグ☁️**

上に行くアルファ、テンポの速いなどブロウラーゲームスナッピーの遅れについてのletの話では **ニフティSmashers**。

一般的に言えば、インターネット上のプレーヤー間に物理的な距離がある場合は常にラグが存在します。 距離が大きいほど、ラグが大きくなります。 開発者がラグを補正して隠すために使用するさまざまなテクニック & 可能な限り最高のエクスペリエンスを実現するために、ラグを覆い隠すこれらのラグ補正手法をいくつか実装しました。 また、世界中のサーバーにソリューションを組み込んでいるため、最も近いプレーヤーを一致させて、ラグを可能な限り最小限に抑えることができます。

それでも、ペースの速い喧嘩屋が必要とする応答性のレベルを考えると、オンラインマルチプレイヤーを立ち上げる際には多少の遅れが予想されます。

進捗状況をできるだけ早くお知らせできることをうれしく思います。 発売後、皆様からのフィードバックをもとに、ラグをできるだけ目立たなくするために全力を尽くします。

あなたはより多くのこれらの技術について学ぶことに興味を持っている場合は、ここではおよそガブリエル・ガンベッタによって偉大な読み取りである **[遅れ補償](https://www.gabrielgambetta.com/client-side-prediction-server-reconciliation.html)**。
