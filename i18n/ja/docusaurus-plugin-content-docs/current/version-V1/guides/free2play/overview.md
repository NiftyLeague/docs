##### **以下の手順に従って認証フローに入る**

1. [ゲームページ](https://niftyleague.com/games)

2. トップバーの接続をタップしてメタマスクウォレットを接続します（または他の優先暗号ウォレット-ほぼすべてをサポートしています！）

3. メタマスクをインストールしていない場合は、 [インストールして](https://metamask.io/)

4. プロンプトが表示されたら、アドレスを確認するサインインダイアログを確認します

5. WebGLまたはデスクトップゲームを選択します

6. 私たちのシステムは、あなたがDEGENを所有しているか、アクティブなDEGENレンタルを持っているかどうかを自動的にチェックします

7. あなたがDEGENを所有している、またはアクティブなDEGENレンタルを持っている場合は、ゲームに参加するように求められます

8. あなたがDEGENを所有していないか、アクティブなDEGENレンタルを持っている場合は、部族を選択してゲームに参加できる無料プレイオプションが表示されます
