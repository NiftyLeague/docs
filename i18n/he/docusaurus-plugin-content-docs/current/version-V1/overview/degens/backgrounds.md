רקעים של DEGEN הוקצו באקראי עם ההטבעה, מה שהביא לרקע משותף או לאחד משלושה רקעים מיוחדים: **נדיר**, **מטא**, או **אגדי**.

מלבד מראה מגניב, הרקע מגיע עם הטבות מיוחדות כמו מראה מכובד במשחק ומכפילי רווחים נוספים.

### מכפילי רווחים

- נדיר: 1.5
- מטא: 3
- אגדי: 6
