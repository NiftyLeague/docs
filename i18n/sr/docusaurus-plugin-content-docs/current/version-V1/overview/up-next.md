Ово омогућава тржишту да одлучи о цени за сваки ДЕГЕН појединачно, истовремено распоређујући закупнине на све стране, јер ће неки опремљени драгоценим предметима, носивим предметима или позадином имати вишу стопу зараде кроз НФТЛ мултипликаторе.
